To include this module in composer.json

```
{
	"repositories": [
		{
			"name": "yamcsha/application",
			"type": "git",
			"url": "git@webserver.local:zf2/zf2-application-module.git"
		}
	],
	"require": {
		"yamcsha/application": "dev-master"
	}
}
```
